Lab: External Command with fields
=================================

Create fields with the following external commands:

**ping4**

* `ping_wrta`
* `ping_crta`
* `ping_wpl`
* `ping_cpl`

**disk**

* `disk_partitions`
* `disk_wfree`
* `disk_cfree`

Go to **Director** -> **Data fields** and change `disk_partitions` to be an Array type.
